# Citizen Connect - Feedback Portal

A modern, clean, civic-themed web application for citizens to provide feedback on government services.

## Features

- **Modern Design**: Clean, civic-themed interface with glassmorphism effects
- **Service Cards**: Horizontal scrolling cards for different government services
- **Feedback Modal**: Comprehensive feedback form with:
  - Service selection dropdown
  - City/district selection with auto-detect
  - Interactive 5-star rating
  - Image upload with preview
  - Anonymous feedback option
  - Submission confirmation animation

## Tech Stack

- React 18
- Vite
- CSS3 with modern features (glassmorphism, animations, gradients)

## Getting Started

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

### Build

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```
